# Contributing to ConformVault TypeScript SDK

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/conformvault-sdk-ts.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run tests: `npm test`
7. Run type check: `npm run typecheck`
8. Run build: `npm run build`
9. Commit and push your changes
10. Open a Pull Request

## Code Style

- Use TypeScript strict mode
- Add JSDoc comments on all exported types and functions
- Write tests for new functionality
- Follow existing code patterns for service implementations

## Reporting Issues

Please open an issue on GitHub with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Node.js version and OS

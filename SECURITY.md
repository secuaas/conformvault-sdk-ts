# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this SDK, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email us at: **security@secuaas.com**

We will acknowledge your report within 48 hours and provide a detailed response within 5 business days.

## Scope

This policy applies to the ConformVault TypeScript SDK. For vulnerabilities in the ConformVault API itself, please contact us at the same email address.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.2.x   | Yes       |
| < 2.2   | No        |
